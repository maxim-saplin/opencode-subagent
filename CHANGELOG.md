# opencode-subagent

## 0.1.0 (MVP)

- Introduce persistent subagent session management via `run_subagent.sh`, `extract_last.sh`, and `search_history.sh`.
- Support name-based addressing with derived titles and append-only session mapping log.
- Add basic model selection semantics (explicit model, env override, dialog-derived default) and structured error hints.
- Provide documentation: `SKILL.md`, `MODELS.md`, `TEST-CASES.md`, `TROUBLESHOOTING.md`, `QUICKSTART.md`.
